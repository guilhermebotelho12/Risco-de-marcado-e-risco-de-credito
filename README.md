# Risco-de-marcado-e-risco-de-credito

Este trabalho tem como objetivo explorar como fatores climáticos, tais como temperatura,
humidade e precipitação, podem influenciar o comportamento do mercado financeiro, especificamente o retorno dos ativos da EDP. Para tal, utilizam-se dois enfoques principais:
a aplicação do modelo CAPM e a previsão de movimentos ascendentes ou descendentes
no preço de fecho das ações, utilizando indicadores técnicos.
